# Technology Stack: Des Moines AI Pulse

## Overview
The "Des Moines AI Pulse" platform leverages a modern and robust technology stack designed for high performance, scalability, and an engaging user experience. The architecture follows a monorepo-like structure, separating frontend and backend concerns while maintaining a unified development environment.

## Frontend
*   **Framework:** React (with Vite for fast development and optimized builds)
*   **Language:** TypeScript (for type safety and improved developer experience)
*   **UI Library:** shadcn-ui (component library built on Tailwind CSS)
*   **Styling:** Tailwind CSS (utility-first CSS framework for rapid UI development)
*   **3D Graphics:** Three.js and React Three Fiber (for interactive 3D cityscapes and engaging visual elements)

## Backend & Database
*   **Platform:** Supabase (an open-source Firebase alternative)
    *   **Database:** PostgreSQL (relational database for data storage and complex queries)
    *   **Edge Functions:** TypeScript/JavaScript (serverless functions for API endpoints, business logic, and integrations)

## Testing
*   **End-to-end Testing:** Playwright (for reliable and fast browser automation and testing)

## Deployment
*   **Platform:** Cloudflare Pages (for fast, secure, and scalable static site and serverless function deployment)

## Scripting/Tooling
*   **Language:** Python (used for various scripting, data processing, and potentially AI/ML related tasks and tooling)
