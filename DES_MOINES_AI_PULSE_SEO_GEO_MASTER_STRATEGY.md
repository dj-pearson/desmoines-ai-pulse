## Part 2: GEO (Generative Engine Optimization) Strategy

### Understanding GEO: The New SEO

**Research Findings**:
- AI searches average 23 words vs Google's 4 words
- 58% of Google searches end without a click (AI Overview)
- GEO methods boost visibility by up to 40% in AI responses
- Relevance > Authority for AI engines (unlike traditional SEO)

### GEO Implementation Framework

#### 1. Statistics Addition Method
**Research Shows**: 115% increase in visibility for 5th-ranked sources

**Implementation**:
```markdown
Before: "Great pizza place in Des Moines"
After: "Serving Des Moines since 1987, this pizzeria has won 12 local awards 
and serves over 5,000 customers weekly with its wood-fired Neapolitan pies."
```

**Action Items**:
- Add quantifiable data to every event (attendance figures, ratings, years running)
- Include restaurant statistics (covers served, awards won, establishment date)
- Cite third-party verification ("According to Des Moines Register...")
- Add comparison data ("One of only 3 authentic Thai restaurants in Des Moines")

#### 2. Quotation Addition Method
**Research Shows**: 37% improvement in subjective impression scores

**Implementation**:
- Add direct quotes from organizers, chefs, attendees
- Include expert testimonials
- Feature review snippets with attribution
- Use "According to..." phrases to cite sources

**Example**:
```markdown
"This is hands down the best farmers market in Iowa," says Sarah Johnson, 
Des Moines Register food critic. "The vendor quality is unmatched."
```

#### 3. Cite Sources Method
**Research Shows**: Most effective for building AI engine trust

**Implementation**:
- Reference official sources (city records, business registrations)
- Link to news articles about venues/events
- Cite social media presence and follower counts
- Include verification badges/certifications

#### 4. Easy-to-Understand Method
**Research Shows**: AI engines prefer content that's easily parseable

**Content Structure**:
```markdown
# [Event Name]: What You Need to Know

## Quick Facts
- When: [Date/Time]
- Where: [Specific Address]
- Cost: [Price Range]
- Who: [Target Audience]

## What to Expect
[2-3 sentence overview]

## Why It Matters
[Local context/significance]

## How to Attend
[Step-by-step instructions]
```

#### 5. Technical Accessibility Method
**Research Shows**: AI engines prioritize fast, well-structured content

**Implementation Checklist**:
- ‚úÖ Schema.org markup (Event, LocalBusiness, Review)
- ‚úÖ JSON-LD structured data on every page
- ‚úÖ Fast load times (<2.5s LCP)

## Part 5: Dynamic Sitemap Implementation

### Current Problem
Sitemap is likely static and may not include:
- Dynamic event pages
- Blog articles (auto-generated)
- New restaurant openings
- Neighborhood pages
- Guide pages

### Solution: Automated Sitemap Generation

#### Edge Function Implementation

**File**: `/supabase/functions/generate-dynamic-sitemap/index.ts`

```typescript
import { serve } from "https://deno.land/std@0.168.0/http/server.ts";
import { createClient } from "https://esm.sh/@supabase/supabase-js@2";

serve(async (req) => {
  try {
    const supabase = createClient(
      Deno.env.get("SUPABASE_URL") ?? "",
      Deno.env.get("SUPABASE_SERVICE_ROLE_KEY") ?? ""
    );

    // Fetch all dynamic content
    const [events, restaurants, attractions, playgrounds, articles, neighborhoods] = 
      await Promise.all([
        supabase.from("events").select("id, date, updated_at").gte("date", new Date().toISOString()),
        supabase.from("restaurants").select("id, updated_at"),
        supabase.from("attractions").select("id, updated_at"),
        supabase.from("playgrounds").select("id, updated_at"),
        supabase.from("articles").select("slug, updated_at, published_at").eq("status", "published"),
        supabase.from("neighborhoods").select("slug, updated_at")
      ]);

    // Build sitemap XML
    const baseUrl = "https://desmoinesaipulse.com"; // Update with actual domain
    
    let sitemap = `<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
        xmlns:news="http://www.google.com/schemas/sitemap-news/0.9"
        xmlns:xhtml="http://www.w3.org/1999/xhtml"
        xmlns:mobile="http://www.google.com/schemas/sitemap-mobile/1.0"
        xmlns:image="http://www.google.com/schemas/sitemap-image/1.1">

  <!-- Homepage -->
  <url>
    <loc>${baseUrl}/</loc>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
    <lastmod>${new Date().toISOString()}</lastmod>
  </url>

  <!-- Hub Pages -->
  <url><loc>${baseUrl}/events</loc><changefreq>daily</changefreq><priority>0.9</priority></url>
  <url><loc>${baseUrl}/restaurants</loc><changefreq>weekly</changefreq><priority>0.9</priority></url>
  <url><loc>${baseUrl}/attractions</loc><changefreq>monthly</changefreq><priority>0.8</priority></url>
  <url><loc>${baseUrl}/playgrounds</loc><changefreq>monthly</changefreq><priority>0.8</priority></url>
  <url><loc>${baseUrl}/guides</loc><changefreq>weekly</changefreq><priority>0.8</priority></url>
  <url><loc>${baseUrl}/articles</loc><changefreq>daily</changefreq><priority>0.8</priority></url>
  <url><loc>${baseUrl}/neighborhoods</loc><changefreq>monthly</changefreq><priority>0.8</priority></url>

  <!-- Dynamic Event Pages -->
`;

    // Add events
    events.data?.forEach(event => {
      sitemap += `  <url>
    <loc>${baseUrl}/events/${event.id}</loc>
    <changefreq>daily</changefreq>
    <priority>0.7</priority>
    <lastmod>${event.updated_at}</lastmod>
  </url>\n`;
    });

    // Add restaurants
    restaurants.data?.forEach(restaurant => {
      sitemap += `  <url>
    <loc>${baseUrl}/restaurants/${restaurant.id}</loc>
    <changefreq>weekly</changefreq>
    <priority>0.7</priority>
    <lastmod>${restaurant.updated_at}</lastmod>
  </url>\n`;
    });

    // Add attractions
    attractions.data?.forEach(attraction => {
      sitemap += `  <url>
    <loc>${baseUrl}/attractions/${attraction.id}</loc>
    <changefreq>monthly</changefreq>
    <priority>0.6</priority>
    <lastmod>${attraction.updated_at}</lastmod>
  </url>\n`;
    });

    // Add playgrounds
    playgrounds.data?.forEach(playground => {
      sitemap += `  <url>
    <loc>${baseUrl}/playgrounds/${playground.id}</loc>
    <changefreq>monthly</changefreq>
    <priority>0.6</priority>
    <lastmod>${playground.updated_at}</lastmod>
  </url>\n`;
    });

    // Add articles
    articles.data?.forEach(article => {
      sitemap += `  <url>
    <loc>${baseUrl}/articles/${article.slug}</loc>
    <changefreq>weekly</changefreq>
    <priority>0.7</priority>
    <lastmod>${article.updated_at}</lastmod>
  </url>\n`;
    });

    // Add neighborhoods
    neighborhoods.data?.forEach(neighborhood => {
      sitemap += `  <url>
    <loc>${baseUrl}/neighborhoods/${neighborhood.slug}</loc>
    <changefreq>monthly</changefreq>
    <priority>0.7</priority>
    <lastmod>${neighborhood.updated_at}</lastmod>
  </url>\n`;
    });

    sitemap += `</urlset>`;

    return new Response(sitemap, {
      headers: {
        "Content-Type": "application/xml",
        "Cache-Control": "public, max-age=3600", // Cache for 1 hour
      },
    });

  } catch (error) {
    return new Response(JSON.stringify({ error: error.message }), {
      status: 500,
      headers: { "Content-Type": "application/json" },
    });
  }
});
```
